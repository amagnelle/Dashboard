# 📊 Dashboard de Análise de Dados

Um dashboard interativo desenvolvido em Python para análise de dados a partir de arquivos CSV.
O projeto permite visualizar métricas, filtrar informações e gerar gráficos dinâmicos de forma simples e intuitiva.

---

## 🚀 Tecnologias utilizadas

* Python
* Streamlit
* Pandas
* Plotly

## 📁 Funcionalidades

* 📤 Upload de arquivos CSV
* 📊 Visualização de dados em tabela
* 📅 Filtro por intervalo de datas
* 💰 Cálculo automático de métricas (total e média)
* 📈 Gráficos interativos (barras e linha)

## 🛠️ Como rodar o projeto na sua máquina

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

---

### 2. Acesse a pasta do projeto

```bash
cd nome-do-projeto
```

---

### 3. Crie um ambiente virtual (recomendado)

```bash
python -m venv venv
```

#### Ativar o ambiente:

**Windows:**

```bash
venv\Scripts\activate
```

**Mac/Linux:**

```bash
source venv/bin/activate
```

---

### 4. Instale as dependências

```bash
pip install -r requirements.txt
```

---

### 5. Execute o projeto

```bash
streamlit run app.py
```

---

### 6. Acesse no navegador

O projeto abrirá automaticamente ou acesse:

```
http://localhost:8501
```
